Installation Guide

  1. Download Anaconda from https://www.anaconda.com/products/distribution and follow the installation instructions (https://docs.anaconda.com/anaconda/install/). Once the installation is complete, make sure to restart your terminal.
  2. Clone the repository:
    git clone https://github.com/hpcclab/E2C-Sim.git
  3. From the E2C-Sim/V1 directory, run the following command to install required packages.
    conda env create -f environment.yml
    (Note: Linux users may need to run conda env export --no-builds > environment.yml prior)
  4. Activate the environment in Anaconda Prompt:
    conda activate simulator
  5. Within the V1 directory, you can run python main.py
  6. To change the config of the machines and task types, open config.json and add new task types in the same format present and increase or decrease the amount of each machine present as well as add new machine types etc also in the format present. Once the desired changes have been made, save the changed files and run python main.py to see the results in the window.
