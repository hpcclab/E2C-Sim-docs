Full Report

The machine report gives information about each machine in the simulation, e.g. m1, m2, m3, etc.. It includes:

  • Machine: The identification for a specific machine.
  • Tasks assigned: How many tasks a given machine was assigned to by the scheduler.
  • Tasks completed: How many tasks a given machine completed successfully.
  • Tasks missed: How many tasks that passed their deadline (if any) on a given machine.
