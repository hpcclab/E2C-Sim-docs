Getting Started

The first step to start the E2C application is to change the working directory to the repository location. Next, you should type the below command in terminal or Anaconda Prompt (Windows) to activate the simulator environment:

conda activate simulator

To access the main.py module, navigate to E2C-Sim/V1/ folder. Then, type the follwoing command in terminal to start the E2C applications:

python main.py

The application interface appears as shown in figure below:


The application interface constitutes of four parttions:

  1. Graphic View : The schematic of the system components are shown here. In addition, processing the workload is animating in this view part. Note that the components in Graphic View are clickable.
  2. Right Dock: Once a user click on a selectable component in Graphic View, the related information is shown in the Right Dock.
  3. Play Control: It contains the control buttons that enable the user to start, pause, or incremental step forward the animation. In additon, the user can use the reset button to reset the system settings to default values.
  4. Menubar: The user can have access to the reports via Report menu. Also, a general information about the E2C can be found in Help menu.
