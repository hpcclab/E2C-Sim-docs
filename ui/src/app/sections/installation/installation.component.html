Installation Guide

  1. Download Anaconda from https://www.anaconda.com/products/distribution and follow the installation instructions (https://docs.anaconda.com/anaconda/install/). Once the installation is complete, make sure to restart your terminal.
  2. Clone the repository:
    git clone https://github.com/hpcclab/E2C-Sim.git
  3. In this step, you create a new environment via conda and name it simulator. For this purpose, open terminal (or Anaconda Prompt for Winodws user) and type the below command:
    conda create --name simulator python=3.9
    Note that the python version in simulator environment is 3.9.
  4. Activate the simulator environment in terminal (or Anaconda Prompt for Winodws users) as follows:
    conda activate simulator
  5. Install the latest version of pip in simulator environment as follows:
    conda install -c anaconda pip
  6. Then, you can use requirements.txt file to install required dependencies via pip:
    pip install -r requirements.txt
  7. Navigate to the E2C-Sim/V1 directory. Then, run the main.py within the V1 directory in simulator environemnt:
    python main.py
  8. To change the config of the machines and task types, open config.json and add new task types in the same format present and increase or decrease the amount of each machine present as well as add new machine types etc also in the format present. Once the desired changes have been made, save the changed files and run python main.py to see the results in the window.
